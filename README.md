# Huff-Pad
허프만 트리를 이용한 텍스트 압축이 가능한 메모장 프로그램

## 주요 기능
- 허프만 트리를 이용한 텍스트 압축 (반복적인 문자가 많이 나오는 문서에 효과가 있습니다.)
- 윈도우의 메모장(Notepad) 기능을 대부분 구현(자동 줄바꿈, 검색/치환, 인쇄, 저장 및 새로 저장 등
- 줄 수, 글자 수 표시 기능
